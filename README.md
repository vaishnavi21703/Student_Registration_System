# Student Registration System

A web-based student registration system developed using Java Servlets and JSP.

## Features
- Student Information 
- Qualification Information
- Address Information
- Display Complete Information

## Technologies Used
- Java
- JSP
- Servlets
- Eclipse
- Apache Tomcat

## Project Structure
- StudentInfo.jsp – Student personal details form
- QualificationInfo.jsp – Academic details form
- AddressInfo.jsp – Address details form
- Display.jsp – Final display page
- Servlets – Handle backend logic and form processing
